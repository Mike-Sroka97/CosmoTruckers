The cosmo-truckers have a starting staff including 
Electricutioner, Saleinator, Cap Cap, and safeT

each zone the cosmo-truckers will have a new unlockable character
The only other currently planned playable character is the CEO of cosmo-trucking

Each player character folder should include the following:

Character name
Character lore
Character visual description / concept art
Character playable classes (DPS, Tank, Support, Utility) [pick 3 duplicates allowed]
Character main mechanics
Character mana base
Character 8 base spells (may reduce to 5)
Character 4 unique spells per skill tree (may increase to 5)
